Apresentação GIT

Fluxo de uso do GIT

  • git clone: copiar um repositório já existente em um diretório local.
  • git add: adicion arquivos à 'staging area'.
  • git commit: adiciona arquivos à 'staging area'.
  • git push (nome do repositório local, nome do repositório remoto): envia os arquivos da 'staging area' para o repositório remoto.

Baixar repositório

  • git pull: adiciona arquivos à 'staging area'.

Mostrar commits

  • git reflog: mostra commits'.

Reverter commits

  • git reset HEAD@{}: reverter commits'.
